# TermSync [용어 통일 모드] 완전판 User Flow

---

## 📖 시나리오: 민지의 카메라 매뉴얼 용어 통일 작업

**배경:**

- 민지는 카메라 회사 기술 문서팀
- 3명의 작가가 각자 작성한 매뉴얼 3개 받음
- 용어가 제각각 (Auto 모드 vs Manual 모드, 셔터 스피드 vs 셔터 속도...)
- TermSync로 한 번에 통일하려고 함

## 🎬 Act 1: 시작

### **Scene 1-1: 랜딩 페이지**

```markdown
민지가 브라우저에서 termsync.app 접속

화면:
┌────────────────────────────────────────────────────────┐
│                                                        │
│                    🔄 TermSync                         │
│                                                        │
│              기술 문서의 용어 통일을                     │
│                  AI가 자동으로                          │
│                                                        │
│     여러 문서에 흩어진 용어들을 AI가 자동으로 분석하고,  │
│           표준 용어로 일관되게 통일해 드립니다           │
│                                                        │
│                  [ 시작하기 → ]                         │
│                                                        │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                        │
│  ⚡ 빠른 분석         📄 다양한 형식      🔒 정확한 통일 │
│  GPT-4 기반 AI      DOCX, PDF 지원     문맥 이해 매칭  │
│  자동 그룹화         최대 10개 문서      신뢰도 표시     │
│                                                        │
└────────────────────────────────────────────────────────┘
```
